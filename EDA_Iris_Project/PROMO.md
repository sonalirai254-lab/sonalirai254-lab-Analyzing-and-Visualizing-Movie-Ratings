## LinkedIn Post

🌸 Just shipped a new Data Science project: **Exploratory Data Analysis on the Iris Dataset** 🌸

As part of my data science journey, I built a complete, end-to-end EDA pipeline on the classic Iris dataset — the dataset every data scientist meets first, and one that still has plenty to teach.

🔍 What's inside:
• Data loading, cleaning & duplicate / missing-value checks
• Descriptive statistics and class-balance analysis
• Correlation heatmap to surface feature relationships
• Outlier detection using the IQR method
• Distribution plots, boxplots, scatter plots and a full pairplot
• Species-wise breakdown of every feature
• A polished **multi-page PDF report** summarizing all findings

💡 Key takeaways:
✔️ Petal length & petal width are the strongest discriminators between species (correlation ~0.96)
✔️ *Setosa* is linearly separable from *versicolor* and *virginica*
✔️ Sepal features alone are not enough to cleanly separate the classes
